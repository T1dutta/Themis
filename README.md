### ⚖️ [Themis — AI Bias Detection Platform](https://github.com/T1dutta/Themis)
> AI-powered platform that analyzes datasets for demographic disparities, explains bias findings in plain English, and provides actionable mitigation steps.

**Stack:** `React 19` `Vite` `Flask` `Pandas` `Fairlearn` `Google Gemini` `Firebase Hosting` `Cloud Functions` `Axios` `PapaParse`

- 📊 Upload any CSV dataset and detect demographic/statistical bias automatically
- 🤖 Gemini AI explains bias findings in plain English with context
- 🛠️ Actionable steps generated to mitigate unfairness in data
- 📈 Fairlearn-powered statistical fairness metrics engine
- ☁️ Fully deployed — Firebase Hosting (frontend) + Cloud Functions (backend)

🌐 **Live Demo:** [themis-179.web.app](https://themis-179.web.app)
